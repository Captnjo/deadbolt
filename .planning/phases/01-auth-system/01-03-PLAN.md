    Create `lib/providers/auth_provider.dart` with the following:

    ```dart
    import 'dart:async';
    import 'package:flutter_riverpod/flutter_riverpod.dart';
    import 'package:shared_preferences/shared_preferences.dart';
    import '../src/rust/api/auth.dart' as auth_bridge;

    enum AuthStatus { locked, unlocked }

    class AuthState {
      final AuthStatus status;
      final int failedAttempts;
      const AuthState({this.status = AuthStatus.locked, this.failedAttempts = 0});

      AuthState copyWith({AuthStatus? status, int? failedAttempts}) {
        return AuthState(
          status: status ?? this.status,
          failedAttempts: failedAttempts ?? this.failedAttempts,
        );
      }
    }
    ```

    The `AuthNotifier` class extends `StateNotifier<AuthState>`:

    - Constructor takes `Ref _ref`, initializes with `AuthState(status: AuthStatus.locked, failedAttempts: 0)`.
    - Private `Timer? _idleTimer`.
    - `static const int kDefaultIdleTimeoutSeconds = 900;` (15 min default).
    - `static const List<int> _delays = [0, 1, 2, 5, 10, 30];` for escalating delay.

    Methods:
    - `int get _timeoutSeconds`: reads from SharedPreferences via `_ref.read(idleTimeoutSecondsProvider)`. Falls back to `kDefaultIdleTimeoutSeconds`.
    - `int delayForAttempt(int attempts)`: returns `_delays[(attempts - 1).clamp(0, _delays.length - 1)]` if attempts > 0, else 0.
    - `Future<void> unlock(String password) async`:
      1. Call `await auth_bridge.verifyAppPassword(password: password)` — this throws on wrong password.
      2. Call `await auth_bridge.unlockApp()` — this unlocks all wallets and sets AtomicBool.
      3. Set state to `AuthState(status: AuthStatus.unlocked, failedAttempts: 0)`.
      4. Call `_resetIdleTimer()`.
    - `void recordFailedAttempt()`:
      Set state to `state.copyWith(failedAttempts: state.failedAttempts + 1)`.
    - `Future<void> lock() async`:
      1. `_idleTimer?.cancel();`
      2. `await auth_bridge.lockApp();` — sets AtomicBool + zeroizes seeds.
      3. Set state to `AuthState(status: AuthStatus.locked, failedAttempts: state.failedAttempts)`.
    - `void resetActivity()`:
      If `state.status == AuthStatus.unlocked`, call `_resetIdleTimer()`.
    - `void _resetIdleTimer()`:
      1. `_idleTimer?.cancel();`
      2. `final timeout = _timeoutSeconds;`
      3. If timeout <= 0 (user chose "Never"), do not start timer. Return.
      4. `_idleTimer = Timer(Duration(seconds: timeout), () { lock(); });`
    - Override `dispose()`: `_idleTimer?.cancel(); super.dispose();`

    Providers:
    ```dart
    final authProvider = StateNotifierProvider<AuthNotifier, AuthState>((ref) {
      return AuthNotifier(ref);
    });

    /// Idle timeout in seconds, stored in SharedPreferences.
    /// 0 means "Never" (no auto-lock).
    final idleTimeoutSecondsProvider = StateProvider<int>((ref) {
      return AuthNotifier.kDefaultIdleTimeoutSeconds;
    });

    /// Initialize idle timeout from SharedPreferences.
    /// Call once at app startup.
    Future<void> initIdleTimeout(WidgetRef ref) async {
      final prefs = await SharedPreferences.getInstance();
      final seconds = prefs.getInt('idle_timeout_seconds') ?? AuthNotifier.kDefaultIdleTimeoutSeconds;
      ref.read(idleTimeoutSecondsProvider.notifier).state = seconds;
    }

    /// Save idle timeout to SharedPreferences.
    Future<void> setIdleTimeout(WidgetRef ref, int seconds) async {
      final prefs = await SharedPreferences.getInstance();
      await prefs.setInt('idle_timeout_seconds', seconds);
      ref.read(idleTimeoutSecondsProvider.notifier).state = seconds;
    }
    ```
