    **PasswordStrength evaluateStrength(String password):**
    - If `password.length < 8`: return `weak`
    - Count score from: `hasUpper` (contains `[A-Z]`), `hasLower` (contains `[a-z]`), `hasDigit` (contains `\d`), `hasSpecial` (contains `[^A-Za-z0-9]`). Score = count of true.
    - If `score >= 3 && password.length >= 12`: return `strong`
    - If `score >= 2`: return `fair`
    - Else: return `weak`

    **class PasswordStrengthMeter extends StatelessWidget:**
    Constructor takes `String password`.

    Build method:
    - `final strength = evaluateStrength(password);`
    - If password is empty, return `SizedBox.shrink()` (no meter shown for empty).
    - Column with:
      - Container height 4px, full width, with `ClipRRect(borderRadius: 2)`:
        - `Stack` with grey background `(BrandColors.border)` and colored foreground:
          - Weak: 33% width, color `BrandColors.warning` (#E2A93B)
          - Fair: 66% width, color `BrandColors.success` (#2ECC71)

        - Use `FractionallySizedBox(widthFactor: fillFraction, ...)` inside the Stack for the colored bar.
      - SizedBox(height: 4)

        - Weak: "Weak" in `BrandColors.warning`
        - Fair: "Fair" in `BrandColors.success`
        - Strong: "Strong" in `BrandColors.primary`
        - Style: `TextStyle(fontSize: 14, fontWeight: FontWeight.w400)` (Label-Hint / regular)
