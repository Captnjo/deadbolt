enum ServerStatus { running, stopped, error }

class AgentServerState {
  final ServerStatus status;
  final String? errorMessage;
  final int? port;
  const AgentServerState({required this.status, this.errorMessage, this.port});
  const AgentServerState.stopped();
  const AgentServerState.running(this.port);
  AgentServerState.error(this.errorMessage);
}

class AgentServerNotifier extends AsyncNotifier<AgentServerState> {
  Future<void> toggleServer(bool enable);
}

class AgentKeyNotifier extends Notifier<List<ApiKeyEntry>> {

**_buildCurlCommand helper:**
```dart
String _buildCurlCommand(List<agent_bridge.ApiKeyEntry> keys) {
  if (keys.isEmpty) return '';
  final keyIndex = _selectedKeyIndex.clamp(0, keys.length - 1);
  final token = keys[keyIndex].tokenMasked;  // Use masked for display
  return 'curl -H "Authorization: Bearer $token" http://localhost:9876$_selectedEndpoint';
}
```

Per the CONTEXT.md decision "curl command with the user's real bearer token", implement the copy action as:
1. On copy button press, call `showAuthChallengeDialog`
2. On auth success, call `ref.read(agentKeyProvider.notifier).getFullKey(keys[keyIndex].tokenPrefix)`
3. Build curl with the real token and copy to clipboard
4. Show snackbar "Copied to clipboard"

**Interaction flows (private methods):**

**_createKey(BuildContext context, WidgetRef ref):**
1. `final authed = await showAuthChallengeDialog(context);` — if false, return
2. Show `showDialog` with AlertDialog:
   - title: "Create API Key"
   - content: TextField with hint "e.g. Claude agent, Trading bot" + "Create" ElevatedButton
   - On Create: `final token = await ref.read(agentKeyProvider.notifier).createKey(label);`
3. Dialog transitions in-place to show:
   - title: "API Key Created"
   - body: "Copy this key now. It will not be shown again."
   - ReadOnly TextField with monospace font showing the full token
   - "Copy & Close" ElevatedButton — copies token, closes dialog

**_revealKey(BuildContext context, WidgetRef ref, ApiKeyEntry key):**
1. `final authed = await showAuthChallengeDialog(context);` — if false, return
2. `final fullKey = await ref.read(agentKeyProvider.notifier).getFullKey(key.tokenPrefix);`
3. Show AlertDialog:
   - title: "API Key"
   - body: "This is your full API key. Keep it secret."
   - ReadOnly monospace TextField with full key
   - "Copy & Close" ElevatedButton

**_confirmRevoke(BuildContext context, WidgetRef ref, ApiKeyEntry key) -> Future<bool>:**
1. Show AlertDialog:
   - title: "Revoke API key?"
   - content: `Text('"${key.label}" will be permanently revoked. Any agent using it will lose access immediately.')`
   - actions: "Keep Key" TextButton (returns false), "Revoke Key" OutlinedButton with BrandColors.error foreground
2. If user chose "Revoke Key": `final authed = await showAuthChallengeDialog(context);`
3. If authed: call `getFullKey(key.tokenPrefix)` then `revokeKey(fullToken)`.
   - If last key revoked, also stop server: `ref.read(agentServerProvider.notifier).toggleServer(false);`
4. Return true (dismiss) or false (cancel)
