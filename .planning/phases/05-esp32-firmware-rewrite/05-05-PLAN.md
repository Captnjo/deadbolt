10. Send `{"cmd":"pubkey"}` — verify pubkey matches generate output
11. Disconnect/reconnect ESP32 — verify pubkey persists (encrypted NVS)
12. Send `{"cmd":"reset"}` — hold BOOT for 5s — verify NVS erased and device reboots clean

**Full integration (Flutter + ESP32):**
13. Pair device, generate keypair, verify mnemonic display + quiz + dashboard flow
14. Verify screenshot prevention on mnemonic screen
