Algorithm pipeline (port from mnemonic.rs):
1. generate(12): 128 bits entropy -> SHA256 checksum (top 4 bits) -> 132 bits -> 12 x 11-bit indices -> BIP39 wordlist lookup

**bip39_wordlist.h:**
- Define `static const char* const BIP39_WORDLIST[] PROGMEM` containing all 2048 BIP39 English words
- Source the wordlist from the BIP-0039 specification (the standard English wordlist)
- Include a `#define BIP39_WORDLIST_SIZE 2048` constant

**bip39.h / bip39.cpp:**

Functions to implement:

1. `bool validateEntropy()`:
   - Call `bootloader_random_enable()` (include `bootloader_random.h` with `extern "C"`)
   - Sample 256 bytes via `esp_fill_random()`
   - Call `bootloader_random_disable()`
   - Run chi-squared test on byte frequency distribution: count occurrences of each byte value (0-255) across the 256 samples, compute chi2 = sum((observed - expected)^2 / expected) where expected = 1.0 (256 samples / 256 possible values), return true if chi2 < 350.0 (p=0.001 threshold for 255 DOF)

2. `void generateMnemonic(char words[12][10])`:
   - Call `bootloader_random_enable()`, fill 16 bytes entropy via `esp_fill_random()`, call `bootloader_random_disable()`
   - Compute SHA-256 of the 16-byte entropy using `mbedtls_sha256(entropy, 16, hash, 0)`
   - Pack entropy[16] + top 4 bits of hash[0] into a 132-bit stream
   - Extract 12 x 11-bit indices from the bit stream
   - Look up each index in `BIP39_WORDLIST[]` and copy into `words[i]`
   - Zeroize entropy buffer with `memset(entropy, 0, 16)` after use

3. `void mnemonicToSeed(const char* mnemonicStr, uint8_t seed[64])`:
   - Concatenate all 12 words with spaces into a single string
   - Use `mbedtls_pkcs5_pbkdf2_hmac()` with:
     - Password: the mnemonic string
     - Salt: "mnemonic" (BIP39 standard, no custom passphrase for v1)
     - Iterations: 2048
     - Hash: MBEDTLS_MD_SHA512
     - Output: 64 bytes into `seed`
   - Include headers: `mbedtls/pkcs5.h`, `mbedtls/md.h`

4. `void slip10Derive(const uint8_t* bip39seed64, uint8_t privkeyOut32[32])`:
   - Master key: `mbedtls_md_hmac(SHA512, key="ed25519 seed"[12 bytes], data=bip39seed64[64 bytes])` -> I[64]
   - Split: key = I[0..32], chain = I[32..64]
   - Derivation path: `{44, 501, 0, 0}` (Solana standard m/44'/501'/0'/0')
   - For each path component: hardened index = component + 0x80000000
     - data[37] = 0x00 || key[32] || big-endian(hIndex)[4]
     - I = HMAC-SHA512(key=chain, data=data)
     - key = I[0..32], chain = I[32..64]
   - Copy final key[32] to `privkeyOut32`
   - Zeroize: memset key, chain, bip39seed64 intermediates to 0

**nvs_crypto.h / nvs_crypto.cpp:**

1. `void deriveEncKey(uint8_t keyOut32[32])`:
   - Get device MAC: `esp_efuse_mac_get_default(mac)` -> 6 bytes
   - Derive AES key: `mbedtls_md_hmac(MBEDTLS_MD_SHA256, key=mac[6], data="deadbolt-nvs-v1"[15 bytes])` -> keyOut32[32]
   - Include: `esp_mac.h` (or `esp_system.h`), `mbedtls/md.h`

2. `bool encryptSeed(const uint8_t* privkey32, uint8_t* outBuf, size_t* outLen)`:
   - Derive AES key via `deriveEncKey()`
   - Generate random 16-byte IV via `esp_fill_random()`
   - AES-256-CBC encrypt the 32-byte private key (PKCS7 padded to 48 bytes) using mbedtls:
     ```c
     mbedtls_aes_context aes;
     mbedtls_aes_init(&aes);
     mbedtls_aes_setkey_enc(&aes, encKey, 256);
     // Pad privkey32 to 48 bytes (32 bytes + 16 bytes PKCS7 padding, pad byte = 0x10)
     uint8_t padded[48];
     memcpy(padded, privkey32, 32);
     memset(padded + 32, 16, 16); // PKCS7 pad
     mbedtls_aes_crypt_cbc(&aes, MBEDTLS_AES_ENCRYPT, 48, iv, padded, ciphertext);
     ```
   - Output format: IV[16] || ciphertext[48] = 64 bytes total
   - Write to outBuf, set *outLen = 64
   - Zeroize encKey, padded buffer after use
   - Include: `mbedtls/aes.h`

3. `bool decryptSeed(const uint8_t* encData, size_t encLen, uint8_t* privkeyOut32)`:
   - Validate encLen == 64
   - Extract IV = encData[0..16], ciphertext = encData[16..64]
   - Derive AES key via `deriveEncKey()`
   - AES-256-CBC decrypt: `mbedtls_aes_crypt_cbc(&aes, MBEDTLS_AES_DECRYPT, 48, iv, ciphertext, plaintext)`
   - Verify PKCS7 padding (last 16 bytes should all be 0x10)
   - Copy first 32 bytes to privkeyOut32
   - Zeroize encKey, plaintext buffer after use
   - Return false if padding verification fails

**unruggable_esp32.ino rewrite:**

2. **Replace `generateNewKeypair()`** with `generateBip39Keypair(char words[12][10])`:
   - Call `validateEntropy()` first; if fails, set LED error pattern (rapid blink 10x at 50ms) and return false
   - Call `generateMnemonic(words)` to get 12 words
   - Build mnemonic string by joining words with spaces
   - Call `mnemonicToSeed(mnemonicStr, bip39Seed)` to get 64-byte seed
   - Call `slip10Derive(bip39Seed, privateKey)` to get 32-byte Ed25519 private key
   - Derive public key: `Ed25519::derivePublicKey(publicKey, privateKey)` (existing)
   - Compute base58 address (existing `base58Encode`)
   - Encrypt private key: `encryptSeed(privateKey, encBuf, &encLen)`
   - Store in NVS: `prefs.begin("unruggable", false); prefs.putBytes("enc_privkey", encBuf, encLen); prefs.putBytes("pubkey", publicKey, 32); prefs.end();`
   - Zeroize: `memset(bip39Seed, 0, 64); memset(mnemonicStr, 0, sizeof(mnemonicStr));`
   - Set `hasKey = true`
   - Return true

3. **Replace `loadKeypair()`** with encrypted version:
   - `prefs.begin("unruggable", true)`
   - Read `enc_privkey` (expect 64 bytes) and `pubkey` (32 bytes)
   - If enc_privkey exists and is 64 bytes: `decryptSeed(encData, 64, privateKey)`
   - Fall back: if old `privkey` key exists (32 bytes, unencrypted), load it, then immediately re-encrypt and store as `enc_privkey`, remove old `privkey` key (migration path)
   - Derive address from pubkey, set `hasKey = true`
   - `prefs.end()`

   - If held 5s: call `generateBip39Keypair(words)`
   - On success: send JSON response with words array:
     ```
     {"status":"ok","words":["word1","word2",...,"word12"],"pubkey":"hex","address":"base58"}
     ```

   `handleFactoryReset()`:

     - `nvs_flash_erase();` (erases entire default "nvs" partition)

   - If cancelled: `sendError("reset_cancelled")`
   - Include: `nvs_flash.h` with `extern "C"` linkage

   `handleEntropyCheck()`:

   - If pass: `sendResponse("ok", "msg", "entropy_valid")`
   - If fail: `sendError("entropy_check_failed")`
