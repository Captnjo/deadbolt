Jito bundle API:
- Endpoint: https://mainnet.block-engine.jito.wtf/api/v1/bundles
- Method: POST JSON body: {"jsonrpc":"2.0","id":1,"method":"sendBundle","params":[["<base64_tx>"]]}
- Response: {"result": "<bundle_id>"}
- Jito tip amount: typically 10,000-100,000 lamports (0.00001-0.0001 SOL)

   - Add the method:
     ```dart
     /// Submit a transaction as a Jito bundle for MEV protection.
     /// Returns the bundle ID on success.
     Future<String> sendJitoBundle(String base64Tx) async {
       final body = jsonEncode({
         'jsonrpc': '2.0',
         'id': 1,
         'method': 'sendBundle',
         'params': [[base64Tx]],
       });
       final response = await http.post(
         Uri.parse(_jitoEndpoint),
         headers: {'Content-Type': 'application/json'},
         body: body,
       );
       if (response.statusCode != 200) {
         throw Exception('Jito bundle failed: ${response.statusCode} ${response.body}');
       }
       final json = jsonDecode(response.body);
       if (json['error'] != null) {
         throw Exception('Jito error: ${json['error']['message'] ?? json['error']}');
       }
       return json['result'] as String;
     }
     ```
