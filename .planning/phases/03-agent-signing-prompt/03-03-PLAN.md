3. Add global auto-show listener for new intents. In the `build()` method, BEFORE the return statement, add a `ref.listen` for `firstPendingIntentProvider`:

```dart
// Auto-show signing prompt when new intent arrives
ref.listen<PendingIntent?>(firstPendingIntentProvider, (prev, next) {
  if (next != null && prev?.id != next.id) {
    // Only auto-show if:
    // 1. App is not locked
    // 2. No modal is currently showing
    final authState = ref.read(authProvider);
    if (authState.status == AuthStatus.locked) return;

    final navKey = ref.read(rootNavigatorKeyProvider);
    final navContext = navKey.currentContext;
    if (navContext == null) return;

    // Check if a modal is currently showing
    final route = ModalRoute.of(navContext);
    if (route != null && !route.isCurrent) return;

    showSigningPrompt(navContext, next.id);
  }
});
```
